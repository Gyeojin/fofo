전체적인 기획과 홈페이지 프로토타입까지 제작한 프로젝트 입니다.

## 주요 특징
* 메인페이지 / 상품 리스트 / 상품 상세페이지 / 리뷰 페이지 작업   
* jQuery 플러그인 사용으로 다양한 슬라이드 구현   
* 리뷰 페이지의 모달창 기능 구현    
<br>

## 사용한 언어
이 사이트를 만드는데 사용한 언어는 다음과 같습니다.

* [PHP]   
* [CSS]   
* [JavaScript]   
* [JQuery]
